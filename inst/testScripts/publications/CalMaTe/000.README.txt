DATA SETS
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
The following data sets are used in these scripts.  Some are
publicly available, whereas some requires certain prior approvals
before the resource allows you to download the data.


DATA SET: TCGA Affymetrix data set

annotationData/
  chipTypes/
    GenomeWideSNP_6/
      GenomeWideSNP_6,Full,na30,hg18,HB20100215.ugp

totalAndFracBData/
  broad.mit.edu_OV.Genome_Wide_SNP_6.12.6.0,ASCRMAv2/
    GenomeWideSNP_6/
      .average-signals-median-mad,<hashcode>.asb
      <sampleName>,total.asb
      <sampleName>,fracB.asb
  broad.mit.edu_OV.Genome_Wide_SNP_6.12.6.0,ASCRMAv2,CMTN/
    GenomeWideSNP_6/
      <sampleName>,total.asb
      <sampleName>,fracB.asb

Availability: This data set is available from the TCGA portal.  It is only available for people with access to the "protected" TCGA data.
